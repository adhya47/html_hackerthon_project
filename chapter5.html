<a href="about.html">About</a>
<a href="contact.html">Contact</a>

💻 Deployment Basics

Deploy your website online using free platforms:

  • GitHub Pages – Host static HTML sites directly from your repository
  • Vercel – Drag & deploy your project folder
  • Netlify – Supports static websites with continuous deployment

💡 Final Tips

  • Keep folder structure clean
  • Use consistent styling across pages
  • Add a navigation bar on every page
  • Test links and images before deployment
