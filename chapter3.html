📄 HTML Boilerplate

The standard HTML boilerplate is the starting template for every HTML page: 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>

</body>
</html>

📄 Div Tag

The <div> tag is a container element used to group content. 

<div>
  <h2>Hello</h2>
  <p>This is a paragraph inside a div.</p>
</div>

📄 Classes & IDs

Use class for multiple elements and id for a unique element: 

<div class="box">This div has a class.</div>

<div id="unique">This div has an id.</div>

📄 Inline Elements

Inline elements only occupy the space their content requires. Common inline elements:

  • <span> – Generic inline container
  • <a> – Hyperlink
  • <b> – Bold text
  • <i> – Italic text
  • <em> – Emphasized text
  • <strong> – Important text
<p>This is a <b>bold</b> and <i>italic</i> text inside a paragraph.</p>
<p>Visit <a href="https://example.com">Example Site</a></p>
<span style="color:red">This text is red using span.</span>

📄 File Path / Linking Files

HTML uses paths to link CSS, JS, or images:

<link rel="stylesheet" href="css/styles.css">
<script src="js/script.js"></script>
<img src="images/pic.jpg" alt="Image">
